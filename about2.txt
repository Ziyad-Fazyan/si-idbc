Panduan Lengkap Sistem SISTEM SEKOLAH
1. PENGENALAN SISTEM
SISTEM SEKOLAH adalah platform manajemen kampus terintegrasi yang menghubungkan seluruh komponen akademik, keuangan, dan administrasi dalam satu sistem terpusat. Sistem ini dirancang untuk memfasilitasi pengelolaan institusi pendidikan tinggi secara digital dan efisien.
Keunggulan Sistem:
•	Terintegrasi: Semua aspek pengelolaan kampus dalam satu platform
•	Multi-User: Mendukung berbagai jenis pengguna dengan hak akses berbeda
•	Real-time: Informasi yang selalu terkini dan sinkron
•	User-friendly: Interface yang mudah dipahami untuk semua kalangan

3. SISTEM LOGIN TIGA TINGKAT
3.1 Portal Admin/Staff
Proses Login:
1.	Input Username: Masukkan username sesuai departemen
o	admin - Web Administrator
o	finance - Departemen Keuangan
o	academic - Departemen Akademik
o	absen - Departemen Absensi
o	support - Departemen Dukungan Teknis
o	musyrif - Departemen Pembimbing
o	site_manager - Pengelola Situs
2.	Input Password: Masukkan password yang telah ditetapkan
3.	Verifikasi: Sistem akan memverifikasi kredensial
4.	Redirect: Diarahkan ke dashboard sesuai departemen
Fitur Keamanan:
•	Session timeout otomatis
•	Password encryption
•	Login attempt limitation
3.2 Portal Dosen
Proses Login:
1.	Input Email: Format email dosen (contoh: dosen.a@example.com)
2.	Input Password: Password pribadi dosen
3.	Autentikasi: Sistem memverifikasi data dosen
4.	Dashboard: Akses ke dashboard pengajar
Hak Akses Dosen:
•	Mengelola jadwal mengajar
•	Kelola absensi mahasiswa
•	Buat dan nilai tugas
•	Lihat feedback mahasiswa
3.3 Portal Mahasiswa
Proses Login:
1.	Input Email: Format email mahasiswa (contoh: mahasiswa.a@example.com)
2.	Input Password: Password pribadi mahasiswa
3.	Validasi: Sistem mengecek status aktif mahasiswa
4.	Dashboard: Akses ke portal akademik mahasiswa
Hak Akses Mahasiswa:
•	Lihat jadwal dan nilai
•	Absensi perkuliahan
•	Akses materi dan tugas
•	Kelola keuangan

8. PROSEDUR PEMULIHAN AKUN
8.1 Lupa Password
1.	Akses Reset Password: Klik "Lupa Password" di halaman login
2.	Input Email: Masukkan email terdaftar di sistem
3.	Verifikasi Email: Cek email untuk link reset password
4.	Create New Password: Ikuti link dan buat password baru sesuai policy
5.	Login Verification: Login dengan kredensial baru
6.	Security Notification: Sistem akan mengirim notifikasi perubahan password
8.2 Account Lockout Recovery
•	Automatic Unlock: Setelah periode cooldown tertentu
•	Admin Intervention: Kontak support untuk unlock manual
•	Security Review: Investigasi penyebab lockout
